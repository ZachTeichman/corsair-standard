What Corsair Standard Does

Corsair Standard provides deterministic formatting compliance analysis for Microsoft Word DOCX resume files. It checks document structure, including margins, tab stops, spacing, section order, font consistency, and typography, against a set of formatting rules.

Accuracy

We use deterministic rule-based analysis to detect formatting issues. While we work to make detection accurate, results may contain false positives or miss certain issues. Formatting analysis is approximate and based on common finance recruiting conventions.
